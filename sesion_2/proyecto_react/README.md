# Proyecto React - Boilerplate

Proyecto React creado con [Vite](https://vite.dev/) siguiendo la [documentación oficial de React](https://es.react.dev/reference/react).

## Stack tecnológico

- **Framework**: React 19
- **Gestión de estado**: Zustand
- **Estilos**: Bootstrap 5
- **Peticiones fetch**: React Query (TanStack Query v5)
- **Testing**: Jest + Testing Library

## Comandos

```bash
# Instalar dependencias
npm install

# Desarrollo (servidor con HMR)
npm run dev

# Build para producción
npm run build

# Vista previa del build
npm run preview

# Tests
npm test

# Linter
npm run lint
```

## Estructura

```
src/
├── store/           # Stores de Zustand
├── __tests__/       # Tests con Jest
├── App.jsx
├── main.jsx
└── index.css
```
