# ¿Que es una skill?

Una Skill es un conjunto de instrucciones en formato Markdown que enseñan al agente de IA cómo realizar una tarea específica. Es como crear un "manual de procedimientos" que el agente puede consultar y seguir automáticamente. Siempre y cuando en el prompt se incluya las palabras clave de la skill.

Instrucciones que el agente osquestador elige automaticamente.
